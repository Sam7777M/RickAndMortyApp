# Swift Rick & Morty App

Full featured universal (iOS, macOS, iPadOS) app showcasing the Rick and Morty API.



## Design

The app has five key area that we will build out step-by-step

1. Characters
2. Locations
3. Episodes
4. Settings
5. Search
